# battleship
Battleship on Django with WebSockets.
Redis, Django should be installed for this project.
Run 'python manage.py runserver' from the 'mysite' directory. Enter 127.0.0.1 in your web browser. Then log in or register a new user. Select an opponent and send him an outgoing request for the game. If the adversary answers your request, the game will begin. Place ships on the field.
